Setup Windows 11 Inside of Linux

June 17, 2025

Download preferred ISO from Microsoft: Download Windows 11

I recommend Windows 11 English International, supposedly less bloat in there. Look at Microwin – Winutil Documentation if you want to go full mode with setting up an clean Windows ISO.

Installing the graphical user interface for QEMU and a GUI Hypervisor:

1

sudo pacman -S quemu-desktop libvirt virt-manager

See man virsh and man qemu for details.

Enable libvirtd:

1

systemctl enable libvirtd.socket

Open Virtual Machine Manager, click the plus icon and create an virtual machine with Windows ISO for example.
